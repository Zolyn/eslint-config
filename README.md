# eslint-config
Personal eslint configuration.

## Usage
### Javascript
```javascript
module.exports = {
    extends: ['@zolyn/eslint-config/js']
}
```

### Typescript
```javascript
module.exports = {
    extends: ['@zolyn']
}
```

### Typescript + Vue
```javascript
module.exports = {
    extends: ['@zolyn/eslint-config/ts/vue']
}
```
